# Easiest Ways to Make Music

## A. All-in-One Web/Mobile Studio
**BandLab**  
[https://www.bandlab.com/](https://www.bandlab.com/)  
- Full studio in browser/phone
- Drag loops, record vocals/instruments
- No driver setup or plugin installation

## B. Single-Window Music Tools
**SunVox**  
[https://warmplace.ru/soft/sunvox/](https://warmplace.ru/soft/sunvox/)  
- Modular synth + pattern sequencer
- Runs on all platforms
- Free desktop version

**Sonic Pi**  
[https://sonic-pi.net/](https://sonic-pi.net/)  
- Live coding environment
- Built-in tutorials
- Accessible design

## C. Simple Recording Editor
**Audacity**  
[https://www.audacityteam.org/](https://www.audacityteam.org/)  
- Record/edit multiple tracks
- Apply effects
- No MIDI required
