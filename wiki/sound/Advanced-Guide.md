# Advanced Music Production Paths

## Advanced DAW Setups
**Ardour + Carla**  
[https://ardour.org/](https://ardour.org/) + [https://kx.studio/Applications:Carla](https://kx.studio/Applications:Carla)  
- Professional DAW + modular plugin host  
- Route instruments/effects between applications  
- Supports LV2/VST/AU plugins  

**BespokeSynth**  
[https://www.bespokesynth.com/](https://www.bespokesynth.com/)  
- Modular patching environment  
- Visual rack-style interface  
- Scripting support for custom modules  

## Open Source Instruments
**Surge XT**  
[https://surge-synthesizer.github.io/](https://surge-synthesizer.github.io/)  
- Hybrid wavetable/subtractive synth  
- 1800+ presets, modular effects chain  

**Dexed**  
[https://asb2m10.github.io/dexed/](https://asb2m10.github.io/dexed/)  
- FM synth emulating Yamaha DX7  
- Patch librarian for hardware  

**Hydrogen**  
[https://hydrogen-music.org/](https://hydrogen-music.org/)  
- Advanced drum sequencing  
- Multi-layer sample support  

## Specialized Tools
**Carla**  
[https://kx.studio/Applications:Carla](https://kx.studio/Applications:Carla)  
- Plugin host for complex racks  
- Supports multiple plugin formats  

**Stochas**  
[https://stochas.org/](https://stochas.org/)  
- Probabilistic sequencer  
- Polyrhythmic pattern generation
