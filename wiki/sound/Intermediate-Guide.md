# Gentler DAWs for Serious Work

**LMMS**  
[https://lmms.io/](https://lmms.io/)  
- Open source FL Studio alternative  
- Pattern-based sequencing  
- Built-in instruments/effects  

**Ardour**  
[https://ardour.org/](https://ardour.org/)  
- Professional open source DAW  
- Audio + MIDI recording/mixing  
- Customizable workflow  

**Zrythm**  
[https://www.zrythm.org/](https://www.zrythm.org/)  
- Modern intuitive interface  
- Strong automation features  
- LV2 plugin support  

**BespokeSynth**
[https://www.bespokesynth.com/](https://www.bespokesynth.com/)
- Modular patching environment
- Visual rack-style interface
- Supports VST plugins

**Code-Based Tools**
- [Sonic Pi](https://sonic-pi.net/) - Live coding music environment
- [Strudel](https://strudel.cc/) - Web-based TidalCycles pattern engine
