# Bio-Debugger

A mental health resource for non-shaming de-escalation techniques.

## What is Bio-Debugger?

The Bio-Debugger is a 3-page visual guide focused on de-escalation and self-regulation strategies. It provides practical tools for managing stress and maintaining mental well-being during development work or any challenging situation.

## Available Formats

The resource is available in multiple formats for accessibility:

- **PDF**: `pdf/bio-debugger-page1.pdf`, `page2.pdf`, `page3.pdf` - Printable format
- **PNG**: `png/bio-debugger-page1.png`, `page2.png`, `page3.png` - High-resolution images
- **SVG**: `svg/bio-debugger-page1.svg`, `page2.svg`, `page3.svg` - Vector graphics (scalable, editable)

## Usage

This resource is designed to be:
- Printed and posted in your workspace
- Kept as a digital reference
- Shared with team members or community

## Philosophy

The Dilla4x project includes this resource as part of our commitment to accessible, inclusive development. Mental health resources should be as available as technical documentation.
